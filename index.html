Embed Script

You can use a simple embed script to showcase the entire category. it will randomly display your active ads.

01

Place the script.

Inject the script in the head of your website.

                                    
<head>
..
<script>
    src="https://shamelessplug.app/embed.js"
    type="module"
    defer
</script>
</head>

02

Copy the Public ID

Copy the Public ID from the category that has all the ads you would like to display. You will find it in the Category tab.

03

Add a special <div>tag to create ad space on your website.

Add a little bit of HTML code to your website. Place it wherever you would like to display ads.
